Use the repo links until each corrected app has its own fresh Render deployment. Do not point Brent & Co. to older deployed URLs that show the wrong app.

## Deploy on Render

Create a Static Site from this repository:

- Repository: `shayreneabee/brent-and-co`
- Branch: `main`
- Build command: leave blank or use `echo "No build needed"`
- Publish directory: `.`

After the site is live, point the future Brent & Co. domain to this Render service.
